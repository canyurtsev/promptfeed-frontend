Design a notification panel UI for a developer community platform.

Requirements:
- Opens as a slide-in panel from the right
- Header: “Notifications”
- Empty state:
  - Icon
  - Message: “You have no notifications yet”
- Prepared layout for future notifications:
  - Upvotes
  - Replies
  - Mentions
- Clean spacing, no visual noise

This panel should feel lightweight and non-intrusive.
